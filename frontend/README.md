# Community Connect - Frontend

Modern React.js frontend with Tailwind CSS for the Community Connect platform.

## Setup Instructions

1. **Install Dependencies**
   ```bash
   cd frontend
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

   The app will be available at `http://localhost:5173`

3. **Build for Production**
   ```bash
   npm run build
   ```

## Features

- ✅ Modern React.js with Vite
- ✅ Tailwind CSS for styling
- ✅ React Router for navigation
- ✅ Axios for API calls
- ✅ Responsive design
- ✅ User authentication
- ✅ Services management
- ✅ Bookings management
- ✅ Reviews system
